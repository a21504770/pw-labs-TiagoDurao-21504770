HTML5 e CSS

HTML, HyperText Markup Language, é uma linguagem de marcadores (markup), não é uma linguagem de programação, utiliza marcadores (tags) para definir os elementos estruturais de uma página da Web.


Elementos Usei Onde Descrição
Header check_circle Página HTML5 Indica o cabeçalho
Nav check_circle Página HTML5 Indica menu de navegação
Main clear Página HTML5 Indica conteúdo principal
Article clear Indica porção do documento que pode ser vista de forma independente
Section check_circle Página HTML5 Especifica uma parte do conteúdo dos elementos
Aside clear Indica anotações
Footer clear Indica rodapé da secção

CSS, Cascading Style Sheets, é um mecanismo para adicionar estilo a um documento web.
